## Comment generation

Generates JSDoc for named functions, classes, methods, interfaces, enums, types, and variables using syntax only.

## Documentation generation

Builds static multi-page documentation with module navigation, symbol cards, search data, and optional quality dashboard data.

## Quality tools

Supports check, drift, lint, fix, coverage badges, JSON output, quality reports, import graph analysis, and GitHub source links.

## Deterministic by design

No AI calls, no network calls during generation, and the same input should produce the same output.
